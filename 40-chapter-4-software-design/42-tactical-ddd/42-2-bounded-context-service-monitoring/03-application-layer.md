#### 4.2.2.3. Application Layer

**Backend:**

### CommandServices
| Clase | Descripción |
| :--- | :--- |
| TelemetryCommandService | Maneja la lógica de ingesta y validación de datos de la capa Edge |

### QueryServices
| Clase | Descripción |
| :--- | :--- |
